# Обнаружение аномальных респондентов в поисковых запросах

## Автор
КОНОПКО, ШАД-111

## Что считается аномалией
Аномалия — это ситуация, когда дневная активность пользователя (daily_ots) по конкретному бренду превышает адаптивный порог, рассчитанный для группы (категория, бренд, день недели).

## Формулы
- `daily_ots = количество_запросов × вес_респондента`
- `порог = медиана + 4.0 × MAD`
- `MAD = медиана(|daily_ots - медиана|)`

## Почему такой подход
- Медиана и MAD устойчивы к выбросам (в отличие от среднего и стандартного отклонения)
- Коэффициент 4.0 обнаруживает только сильные выбросы
- Группировка по дням недели учитывает разную активность в выходные и будни
- Минимум 5 наблюдений защищает от редких брендов
